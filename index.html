Low-Rank Adaptation (LoRA)

🔧
Freeze & Inject Strategy

Keep your pretrained weights W intact while adding two small matrices:

A ∈ ℝd×r & B ∈ ℝr×k
where r ≪ d, k
W
+
A
B
📐
Update Rule
W' = W + (α/r) · A · B

This elegant formula shows how LoRA updates work: the original weights plus a scaled low-rank update.

⚙️
Key Hyperparameters
Rank (r)

Controls capacity vs. parameter count trade-off

Scale (α)

Determines update strength

Dropout (p)

Regularizes on limited data

Quantization Theory

🔋
Why Quantize?

Dramatically reduce memory usage and computation by using lower-bit formats:

FP16

Half precision floating point

INT8

8-bit integer quantization

INT4

Extreme 4-bit quantization

📊
Uniform Mapping
q = clip(round(x/S) + Z)
x̂ = (q - Z) · S

This mapping converts floating-point values to quantized integers and back.

🎯
Quantization Approaches
Post-Training Quantization (PTQ)

Fast approach with no retraining required

Quantization-Aware Training (QAT)

Simulates quantization during training for minimal accuracy loss
